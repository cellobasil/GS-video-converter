**Описание проекта:**
GS Video Converter Bot - Python приложение, которое обрабатывает медиафайлы из Telegram, сжимает видео и публикует их в канал. Проект уже готов и находится в репозитории: https://github.com/cellobasil/GS-video-converter

1. **Системные ресурсы:**

   - Настройка systemd service для автозапуска бота

   - Логирование в `/var/log/gs-bot/` (или другую директорию)

4. **Для разработки:**
   - SSH доступ для подключения Cursor Agent (для удаленной разработки)

## Дополнительные вопросы, которые могут возникнуть

Если разработчик спросит про безопасность:
- Бот работает только с авторизованными пользователями (список ID в конфиге)
- Не хранит чувствительные данные (только временные файлы)
- Все ключи в `.env` файле (не коммитятся в git)

Если спросит про нагрузку:
- Обработка видео может быть ресурсоемкой, но происходит по требованию
- Временные файлы удаляются после обработки
- Бот работает асинхронно, использует очередь задач

Если спросит про мониторинг:
- Логи пишутся через стандартный Python logging
- Можно настроить отправку логов в централизованную систему (если есть)
